โปรเจกต์วิเคราะห์สุขภาพสวนกล้วยด้วย Sentinel-2

หัวข้อ:
การประเมินสุขภาพสวนกล้วยเชิงพื้นที่ด้วยดัชนีพืชพรรณจากข้อมูลดาวเทียม Sentinel-2
(Spatial Evaluation of Banana Plantation Health Using Vegetation Indices from Sentinel-2)

วัตถุประสงค์:
[1] เพื่อพัฒนากระบวนการจัดเก็บข้อมูลดาวเทียม Sentinel-2 ผ่าน Google Earth Engine (GEE) 
[2] เพื่อคำนวณดัชนีพืชพรรณ NDVI, NDMI และ NDRE ร่วมกับข้อมูลสภาพอากาศสำหรับวิเคราะห์ความสมบูรณ์ของสวนกล้วย
